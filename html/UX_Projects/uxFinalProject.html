Proposal {

This proposal was centered around how I would implement the 5 stages of the UX Design Thinking Process: Empathize, Define, Ideate, Prototype, and Test. Because there was no specific data included in the project assignment, I was unable to draft a proposal target to specific needs, but rather created a proposal to subjectively increase user efficiency, effectiveness, and satisfaction.

Prototype {

I then created a high-fideliity prototype designed to implement the proposed changes. Due to the quick timeline of this project some of the features are not entirely implemented.

The prototype contains several comments to illustrate:

  1. Consistency with the changes proposed in the paper
  2. Demonstrate improvements in efficiency, effectiveness, and satisfaction
  3. Show how each of the usability heuristics are implemented
  4. Show how several user experience laws are implemented

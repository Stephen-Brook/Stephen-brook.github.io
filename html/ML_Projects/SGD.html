Initial implementation of Stochastic Gradient Descent {

This stage started with implementing the update function, with a single dimensional input. In other words, one x to one y.

I decided that the best implementation would use a small data set, that I could easily understand and quickly iterate over. In this case, I chose to use a dataset in the real estate market. For the initial implementation, this model is designed to take in an input variable of square footage, and output an estimated price.

After implementing this algorithm, the program would produce an output in the form of y=mx+b, which I then plotted against the dataset to see how accurate the function is.

Final Implementation {

After testing my algorithm on the smaller dataset, and feeling reasonably confident in its results, I decided to use a different, much larger dataset with additional input variables.

Initially, the algorithm did not funciton as expected, but after normalizing the data, and abstracting the prediction function, I got a result more in line with what I expected.

The attatched image shows the initial data in green, with a line of best fit calculated by desmos also in green, while the predicted calculations are shown in red.
